#####8 iOS与Unity数据通信
8.1 iOS调用Unity方法
iOS调用此函数 UnitySendMessage(<#const char obj#>, <#const char method#>, <#const char *msg#>)
第一个参数：C#对象名
第二个参数：需要调用的方法名
第三个参数：方法传的参数
实例： UnitySendMessage(“GM”, “MoveControllerInIos”, jsonString.UTF8String);
8.2 Unity调用iOS方法
在某个类里实现如下方法。此类.m必须改成.mm 

extern "C" {
    // u3d退出
    void _UnityQuit() {
    }
}
